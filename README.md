# JS Mini Projects

A collection of three vanilla JavaScript mini-apps — each self-contained in a single HTML file.

---

## Projects

| File | Description |
|------|-------------|
| `calculator.html` | A functional calculator with arithmetic operations |
| `rock-paper-sciser.html` | Rock Paper Scissors game against the computer |
| `todolist-Project.html` | To-do list with add, complete, and delete functionality |

---

## Getting Started

No build step needed. Open any `.html` file directly in your browser.

---

## Tech Stack

HTML · CSS · Vanilla JavaScript (all inline, zero dependencies)
